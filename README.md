# japanese-sock-puppet-accounts-on-twitter-list
Research dataset of sock puppet accounts.

## 概要
Twitter(X)上で協調した日本語ソックパペット投稿をおこなっている可能性が高いアカウントのユーザーID一覧です。

## Outline
This is a list of user IDs for accounts highly likely to be engaging in coordinated Japanese-language sockpuppet posting on Twitter (now X).

## Contents
UserID

### ユーザーIDとは？
Twitter(X)の内部ユーザーIDを意味します。「@」から始まるスクリーンネームではありません。

### What is user IDs?
This refers to the internal user ID on Twitter (now X). It is not the screen name that starts with '@'.

## 留意事項
同一もしくは類似の投稿を協調した複数IDからおこなう行為や他人のポスト内容を無作為にコピーしたものの投稿を繰り返す行為をソックパペット投稿とみなし、これらの行為をおこなうアカウントをソックパペットアカウントとしてリスト化しています。
既に削除されたアカウントやX社により停止されたアカウントを含みます。
このデータは公開情報をもとに収集し、研究・分析目的で公開しています。
特定アカウントの自動化、組織的運用、スパム行為、その他不正行為等を断定するものではありません。

## Important Matters
I classify the act of making identical or similar posts from multiple coordinated accounts —or repeatedly posting content indiscriminately copied from others— as "sockpuppet posting," and we maintain a list of the accounts engaging in such behavior as "sockpuppet accounts."
This list includes accounts that have already been deleted or suspended by X.
This data is compiled from publicly available information and is published for research and analysis purposes.
This list does not definitively identify specific accounts as engaging in automation, organized operation, spamming, or other forms of misconduct.
